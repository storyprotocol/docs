---
title: Search IP Assets
description: Perform vector search for IP assets based on query text and optional media type filter. This endpoint uses AI-powered search to find relevant assets by semantic similarity.
openapi: "https://api.storyapis.com/api/v4/openapi.json POST /search"
---

<Note>

Search is done by vectorizing the title and description of the IP metadata using the [IPA metadata standard](https://docs.story.foundation/concepts/ip-asset/ipa-metadata-standard).

</Note>
