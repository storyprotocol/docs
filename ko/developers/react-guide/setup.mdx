특별한 React SDK는 없지만 **React에서 TypeScript SDK를 사용할 수 있습니다** Metamask와 같은 JSON-RPC 계정으로 트랜잭션 서명과 전송을 지연시키기 위해서입니다.

우리는 [wagmi](https://wagmi.sh/)를 Web3 제공자로 사용하고 Dynamic이나 RainbowKit과 같은 지갑 제공자를 설치하는 것을 추천합니다. 다음 모든 것에 대한 예시를 제공합니다:

* [Dynamic 설정](/developers/react-guide/wallet-providers/dynamic-setup)
* [RainbowKit 설정](/developers/react-guide/wallet-providers/rainbowkit-setup)
* [Reown (WalletConnect) 설정](/developers/react-guide/wallet-providers/reown-setup)
* [Tomo 설정](/developers/react-guide/wallet-providers/tomo-setup)
