---
title: LicenseToken
---

LicenseToken 컨트랙트는 LNFT(License NFT)라고도 하며, Story 생태계 내의 IP 자산에 대한 라이선스 계약을 나타내는 ERC721 토큰입니다. 이는 프로그래밍 가능한 IP 라이선스의 생성, 이전 및 관리를 가능하게 합니다.

## 상태 변수

### LICENSE\_REGISTRY

```solidity
ILicenseRegistry public immutable LICENSE_REGISTRY
```

프로토콜 전체 License Registry의 주소입니다.

### LICENSING\_MODULE

```solidity
ILicensingModule public immutable LICENSING_MODULE
```

프로토콜 전체 Licensing Module의 주소입니다.

### DISPUTE\_MODULE

```solidity
IDisputeModule public immutable DISPUTE_MODULE
```

프로토콜 전체 Dispute Module의 주소입니다.

### MAX\_COMMERCIAL\_REVENUE\_SHARE

```solidity
uint32 public constant MAX_COMMERCIAL_REVENUE_SHARE = 100_000_000
```

최대 로열티 비율은 100\_000\_000으로, 100%를 나타냅니다.

### LicenseTokenMetadata

```solidity
struct LicenseTokenMetadata {
    address licensorIpId;
    address licenseTemplate;
    uint256 licenseTermsId;
    bool transferable;
    uint32 commercialRevShare;
}
```

라이선스 토큰의 메타데이터 구조:

* `licensorIpId`: 라이선스 제공자인 IP 자산
* `licenseTemplate`: 라이선스 템플릿 컨트랙트 주소
* `licenseTermsId`: 라이선스 조건의 ID
* `transferable`: 라이선스 토큰의 이전 가능 여부
* `commercialRevShare`: 상업적 수익 공유 비율

## 함수

### initialize

```solidity
function initialize(address accessManager, string memory imageUrl) public initializer
```

LicenseToken 컨트랙트를 초기화합니다.

**Parameters:**

* `accessManager`: 접근 관리자의 주소
* `imageUrl`: 라이선스 토큰의 기본 이미지 URL

### setLicensingImageUrl

```solidity
function setLicensingImageUrl(string calldata url) external restricted
```

모든 라이선스 토큰의 라이선싱 이미지 URL을 설정합니다.

**Parameters:**

* `url`: 라이선싱 이미지의 URL

### mintLicenseTokens

```solidity
function mintLicenseTokens(
    address licensorIpId,
    address licenseTemplate,
    uint256 licenseTermsId,
    uint256 amount,
    address minter,
    address receiver,
    uint32 maxRevenueShare
) external onlyLicensingModule returns (uint256 startLicenseTokenId)
```

지정된 수량의 라이선스 토큰(LNFT)을 발행합니다.

**Parameters:**

* `licensorIpId`: 라이선스 토큰이 발행되는 라이선스 제공자 IP의 ID
* `licenseTemplate`: License Template의 주소
* `licenseTermsId`: License Terms의 ID
* `amount`: 발행할 라이선스 토큰의 수량
* `minter`: 발행자의 주소
* `receiver`: 발행된 라이선스 토큰을 받는 수신자의 주소
* `maxRevenueShare`: 라이선스 토큰 발행에 허용된 최대 수익 공유 비율

**Returns:**

* `startLicenseTokenId`: 발행된 라이선스 토큰의 시작 ID

### burnLicenseTokens

```solidity
function burnLicenseTokens(address holder, uint256[] calldata tokenIds) external onlyLicensingModule
```

주어진 토큰 ID에 대한 라이선스 토큰(LT)을 소각합니다.

**Parameters:**

* `holder`: 라이선스 토큰 보유자의 주소
* `tokenIds`: 소각할 라이선스 토큰의 ID 배열

### validateLicenseTokensForDerivative

```solidity
function validateLicenseTokensForDerivative(
    address caller,
    address childIpId,
    uint256[] calldata tokenIds
) external view returns (
    address licenseTemplate,
    address[] memory licensorIpIds,
    uint256[] memory licenseTermsIds,
    uint32[] memory commercialRevShares
)
```

파생 IP 등록을 위한 라이선스 토큰을 검증합니다.

**Parameters:**

* `caller`: 주어진 토큰으로 파생물을 등록하는 호출자의 주소
* `childIpId`: 파생 IP의 ID
* `tokenIds`: 검증할 라이선스 토큰의 ID 배열

**Returns:**

* `licenseTemplate`: 라이선스 토큰과 연관된 License Template의 주소
* `licensorIpIds`: 각 라이선스 토큰과 연관된 라이선스 제공자 IP 배열
* `licenseTermsIds`: 각 검증된 라이선스 토큰과 연관된 License Terms 배열
* `commercialRevShares`: 각 라이선스 토큰과 연관된 상업적 수익 공유 비율 배열

### totalMintedTokens

```solidity
function totalMintedTokens() external view returns (uint256)
```

처음부터 발행된 총 라이선스 토큰 수를 반환합니다. 이 숫자는 라이선스 토큰이 소각되어도 감소하지 않습니다.

**Returns:**

* `uint256`: 발행된 총 라이선스 토큰 수


**Parameters:**

* `licensorIpId`: 라이센서 IP의 ID.

**Returns:**

* `uint256`: 라이센서 IP에 대해 발행된 라이선스 토큰의 총 수.

### isLicenseTokenRevoked

```solidity
function isLicenseTokenRevoked(uint256 tokenId) public view returns (bool)
```

라이선스가 취소되었는지 여부를 반환합니다(분쟁 모듈에서 분쟁 후 라이센서 IP가 태그된 경우). 태그가 제거되면 라이선스는 더 이상 취소되지 않습니다.

**Parameters:**

* `tokenId`: 라이선스 토큰의 ID.

**Returns:**

* `bool`: 라이선스가 취소된 경우 True.

### tokenURI

```solidity
function tokenURI(uint256 id) public view virtual override(ERC721Upgradeable, IERC721Metadata) returns (string memory)
```

LNFT 매개변수의 ERC721 OpenSea 메타데이터 JSON 표현.

**Parameters:**

* `id`: 라이선스 토큰의 ID.

**Returns:**

* `string`: 라이선스 토큰의 메타데이터 URI.
