# 개요

Story Network는 EVM과 Cosmos SDK의 장점을 모두 달성하는 목적 지향적인 레이어 1 블록체인입니다. IP와 같은 복잡한 데이터 구조를 빠르고 비용 효율적으로 처리하기 위해 설계된 그래프 데이터 구조를 지원하는 심층 실행 레이어 최적화와 함께 100% EVM 호환성을 제공합니다. 이는 다음과 같은 방식으로 구현됩니다:

* IP 그래프와 같은 복잡한 데이터 구조를 최소한의 비용으로 몇 초 안에 탐색할 수 있는 미리 컴파일된 프리미티브 사용
* 빠른 완결성과 저렴한 트랜잭션을 보장하는 성숙한 CometBFT 스택 기반의 합의 레이어
* Ethereum의 Engine-API를 통해 합의와 실행을 분리하는 모듈식 아키텍처
