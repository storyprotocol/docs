# VRF

Gelato VRF는 Drand라는 탈중앙화되고 신뢰할 수 있는 난수 소스를 활용하여 블록체인 애플리케이션을 위한 검증 가능한 무작위성을 제공합니다. 개발자들이 입증 가능하고 변조 방지된 진정한 무작위 값을 받을 수 있도록 보장합니다.

Gelato의 [문서](https://docs.gelato.network/web3-services/vrf) 가이드를 참조하여 가격 피드로 애플리케이션을 통합하세요.
