# Story Network (L1)

Story Chain에 오신 것을 환영합니다.

이 섹션은 Story Network의 기본 사항을 이해하는 데 도움이 되도록 설계되었습니다. 내용을 두 부분으로 구성했습니다:

1. 아키텍처 이해하기
2. 노드 운영하기

Story Network는 이더리움 가상 머신(EVM)과 Cosmos SDK의 장점을 원활하게 통합한 목적 지향적 레이어 1 블록체인입니다. EVM과 완벽하게 호환되면서도 그래프 기반 데이터 구조를 효율적으로 지원하기 위한 심층적인 실행 레이어 최적화를 포함하고 있습니다. 이러한 최적화를 통해 복잡한 지적 재산권(IP) 데이터 구조를 비용 효율적이고 확장 가능한 방식으로 처리하는 데 특히 적합합니다.

## 주요 기능

* **EVM 호환성**: 이더리움 가상 머신과의 완벽한 호환성
* **최적화된 데이터 구조**: IP 그래프 탐색을 위한 사전 컴파일된 프리미티브
* **빠른 완결성**: 신속한 트랜잭션 완결성을 위한 CometBFT 기반 합의 레이어
* **모듈식 아키텍처**: 이더리움의 Engine-API를 사용한 실행에서 합의 분리
