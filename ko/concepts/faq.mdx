## *"License Token, Royalty Token, Revenue Token의 차이점은 무엇인가요?"*

|           | License Tokens                                                                                                                                                                      | Royalty Tokens                                                                                                                                | Revenue Tokens                                                                                                     |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **모듈**    | [📜 라이선싱 모듈](/concepts/licensing-module)                                                                                                                                            | [💸 로열티 모듈](/concepts/royalty-module)                                                                                                         | [💸 로열티 모듈](/concepts/royalty-module)                                                                              |
| **설명**    | 특정 라이선스 조건이 포함된 IP Asset에서 발행되는 ERC-721 NFT입니다. 이는 본질적으로 License Token에 포함된 조건에 따라 관련 IP Asset을 사용할 수 있는 권한을 부여하는 라이선스입니다.<br /><br />License Token은 다른 IP Asset의 파생물로 등록될 때 소각됩니다. | 각 IP Asset에는 100,000,000개의 Royalty Token이 연결되어 있으며, 각 토큰은 소유자가 IPA의 Royalty Vault에 예치된 수익("*Revenue Tokens*")의 0.000001%를 청구할 수 있는 권리를 나타냅니다. | 이는 실제 지불에 사용되는 토큰입니다(예: \$WIP).<br /><br />"*Royalty Tokens*"는 IP Asset이 수익을 얻을 때 이러한 Revenue Token을 청구하는 데 사용됩니다. |
