## 프로그래머블 IP 라이선스(PIL): 법적 집행의 기초

핵심적으로, 모든 [IP Asset](/concepts/ip-asset)은 Story에 등록될 때 **법적 구속력이 있는 문서인 [Programmable IP License (PIL)](/concepts/programmable-ip-license)**&#xB85C; 감싸집니다. 미국 저작권법을 기반으로 하는 PIL은 IP 소유자가 자산에 맞춤형 조건을 첨부할 수 있게 해주는 범용 라이선스 계약 템플릿으로 작용합니다.

**Story에서의 라이선싱은 진정한 법적 약속을 의미합니다.** PIL에 정의된 매개변수—상업적 사용, 파생물 허용, 저작자 표시 요구사항, 로열티 구조—는 IP 소유자(라이선서)와 IP를 라이선스하는 모든 사람(라이선시) 사이의 법적으로 집행 가능한 조건을 나타냅니다.

### PIL은 어떻게 집행을 가능하게 하나요?

* **명확한 법적 조건:** PIL은 IP 소유자가 사용 규칙을 정의하는 표준화된 방법을 제공합니다.
* **온체인 기록을 증거로:** IP Asset에 첨부된 PIL 조건은 Story의 목적 지향적 블록체인에 변경 불가능하게 기록되어 *반박할 수 없는 증거*로 작용합니다.
* **오프체인 법적 구제:** IP가 PIL 조건을 위반하여 오용되는 경우, IP 소유자는 **오프체인 법적 절차**에서 온체인 증거를 활용할 수 있습니다.
* **권리 증명으로서의 라이선스 토큰:** 라이선시는 [**License Token**](/concepts/licensing-module/license-token) (NFT)을 받게 되며, 이는 PIL 조건에 따라 부여된 특정 사용 권한을 나타내고 승인 상태에 대한 추가 증거를 제공합니다.

## Story Attestation Service (SAS): 사전 예방적 침해 모니터링

법적 프레임워크를 넘어서, 우리는 [**Story Attestation Service (SAS)**](/concepts/story-attestation-service)를 구축하여 다층적 분산 접근 방식을 사용하여 IP 소유자가 잠재적인 저작권 침해를 모니터링할 수 있도록 돕고 있습니다.

<Note>
  SAS는 판단 계층이 아닌 신호 계층입니다—자동화된 집행 조치를 취하는 대신 IP 소유자가 조치를 취할 수 있는 잠재적 문제를 표시합니다.
</Note>

### SAS가 침해 탐지를 돕는 방법:

* **전문화된 제공자 네트워크:** SAS는 Yakoa와 Pex와 같은 서비스 제공자들과 협력하여 AI와 머신러닝을 사용해 인터넷과 다른 블록체인에서 다양한 미디어 유형의 저작권 위반을 감지합니다.
* **투명한 신호:** SAS는 제공자의 발견 사항을 기반으로 IP Asset의 정당성에 관한 공개적으로 접근 가능한 신호를 제공합니다.
* **상업적 IP 중심:** 현재 SAS는 주로 상업적 IP Asset에 대한 침해 검사를 실행합니다—최소한 하나의 라이선스 조건이 있는 자산`commercialUse = true`.
* **메타데이터 기반 검사:** SAS는 등록 시 제공된 IP 특정 메타데이터를 사용하여 기존 온라인 콘텐츠에 대한 검사를 수행합니다.

### 중요 고려사항:

* **탐지, 예방이 아님:** SAS는 주로 IP 등록 후 잠재적 침해를 감지하며 이를 예방하지는 않습니다.
* **인터넷 기반 확인:** 현재 SAS는 주로 온라인에 이미 존재하는 콘텐츠를 기반으로 침해를 탐지하며, 오프라인 사용은 탐지하지 않습니다.
* **완벽한 보장 없음:** 어떤 시스템도 모든 저작권 침해를 100% 탐지할 수는 없습니다.

## Dispute Module의 역할

우리는 또한 [**Dispute Module**](/concepts/dispute-module)을 구축했습니다. 이를 통해 누구나 부적절한 등록이나 잠재적 표절과 같은 이유로 IP 자산에 대해 온체인 분쟁을 제기할 수 있습니다. 이는 분쟁이 있는 IP에 대한 온체인 플래깅으로 이어질 수 있으며, 라이선스 생성이나 수익 창출 능력에 영향을 미칠 수 있습니다.

## 하이브리드 집행 모델

<Note>
  Story는 법원이나 변호사를 대체하지 않습니다—온체인 자동화, 투명성, 상호운용성의 이점을 누리면서 전통적인 집행 시스템과 함께 작동하는 도구를 IP 보유자에게 제공합니다.
</Note>

### Story가 할 수 있는 것:

* PIL을 통한 법적으로 유효한 IP 라이선싱 프레임워크 제공
* IP 소유권과 라이선스 조건에 대한 변경 불가능한 온체인 기록 생성
* SAS를 통해 잠재적인 온라인 침해를 탐지하는 모니터링 도구 제공
* Dispute Module을 통한 온체인 분쟁 해결 촉진
* 오프체인 법적 집행에 사용할 수 있는 증거 제공

### Story가 할 수 없는 것:

* IP 침해에 대한 글로벌 경찰력으로 작용
* 모든 무단 IP 사용 방지 보장
* 물리적 세계에서 법적 판결 직접 집행
* 등록된 IP와의 모든 디지털 및 물리적 상호작용 모니터링
